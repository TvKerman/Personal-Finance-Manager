# Personal Finance Manager

Консольное приложение для управления личными финансами с поддержкой нескольких пользователей, бюджетирования и аналитики.

## Функциональность

### Основные возможности
- Регистрация и авторизация пользователей
- Управление доходами и расходами
- Бюджетирование по категориям
- Переводы между пользователями
- Детальная статистика и отчеты
- Система оповещений
- Экспорт данных в CSV и JSON
- Валидация ввода

### Дополнительные возможности
- Предупреждения при достижении установленного лимита бюджета
- Редактирование и удаление бюджетов
- Подсчет расходов по выбранным категориям
- Автоматическое сохранение данных

## Технологии
- Java 22
- Maven
- JUnit 5
- Checkstyle
- GitHub Actions

## Быстрый старт

### Внимание
- Возможно неправильное отображение символов кириллицы в зависимости от системы

### Сборка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/TvKerman/Personal-Finance-Manager.git
cd personal-finance-manager

# Сборка проекта
mvn compile

# Запуск приложения
mvn exec:java

# Запуск тестов
mvn test

# Проверка стиля кода
mvn checkstyle:check
